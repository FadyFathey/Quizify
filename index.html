‚úì Which HTML element is used to define the title of a document?

A The <head> element
B The <title> element
C The <header> element
D The <meta> element
Explanation:

The <title> element defines the title of the HTML document which appears in the browser's title bar or tab. The <head> element is a container for metadata, the <header> is used for introductory content or navigation, and <meta> provides metadata about the HTML document.

‚úó Which attribute is used to specify an alternate text for an image?

A src
B title
C alt
D href
Explanation:

The alt attribute specifies an alternate text for an image, which is displayed if the image cannot be loaded or for accessibility purposes with screen readers. The src attribute specifies the URL of the image, title provides additional information displayed as a tooltip, and href is used for links, not images.
